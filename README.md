# AI 全能专家取名系统

基于 Claude Agent SDK 构建的多代理智能取名系统。通过模拟 8 位不同领域专家的研讨会，为新生儿提供专业、全面的取名建议。

## 功能特点

- **多专家协同**：8 位虚拟专家从不同角度分析名字
- **三轮研讨流程**：提名 → 质询 → 决选，层层筛选
- **结构化输出**：JSON 格式的详细报告，包含评分和点评
- **用户参与**：支持用户在提名阶段追加自己的名字创意
- **自动重试**：流程中断时自动恢复继续

## 专家团队

| 专家 | 职责 |
|------|------|
| 语言学家 | 检查平仄、声韵搭配、谐音风险 |
| 教育心理顾问 | 评估名字对性格发展的影响 |
| 诗词专家 | 确保名字有典故出处和文化底蕴 |
| 父母代表 | 从家长角度审视情感寄托 |
| 星座大师 | 根据出生日期分析星座特质 |
| 算命大师 | 依据五行八字提供建议 |
| 职业规划专家 | 评估名字的职业气场 |
| 网络达人 | 检查网络舆论风险和谐音梗 |

## 安装

```bash
# 克隆项目
git clone <repository-url>
cd naming_system_v1

# 安装依赖
pip install -r requirements.txt
```

## 配置

创建 `.env` 文件并配置 API 密钥：

```env
ANTHROPIC_API_KEY=your_api_key_here
```

## 使用方法

```bash
python main.py
```

运行后按提示输入：
1. **姓氏**：如 "李"
2. **性别**：男孩/女孩
3. **出生信息**：如 "2024年5月20日 早上8点"
4. **期望**：如 "希望聪明、健康，避免生僻字"

系统会自动开始三轮研讨：
- **第一轮（提名）**：各专家提出候选名字，此时可追加自己的名字
- **第二轮（质询）**：所有专家对候选名字打分点评
- **第三轮（决选）**：汇总分数，生成最终排名
